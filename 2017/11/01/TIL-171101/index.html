TIL_171101 Sass 변환 적용

npm sass

폴더구조

  • grid (root)
    • css/
    • sass/
      • partials/
        • grid.scss
      • stylesheet.scss

파일 하나만 css 변환 시 ‘>’사용

폴더 전체를 변환시 -o (output)사용

만약 stylesheet.scss파일에 @import “./partials/grid”;로 호출한 것을 변환 시에는 import가 아닌 들어와있는 상태로 변환하고 싶을 때는, 파일명에 (underscore)를 붙여줌.
단, 쉘 명령어에는 빼고 파일명을 기입한다.
